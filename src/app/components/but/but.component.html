Chaque carte a plusieurs caractéristiques :

couleur, nombre, forme
(et remplissage si niveau avancé)

Le but du jeu est de sélectionner 3 cartes dont chaque caractéristique est soit pareille pour les 3, soit différente pour les 3.

"JAMAIS 2 CONTRE 1"
Lorsque vous avez trouvé un SET et que le signal est vert : BUZZEZ!

Vous avez alors 10 secondes pour cliquer sur 3 cartes.

En cas d'erreur, vous n'aurez pas le droit de buzzer pendant 15 secondes.
